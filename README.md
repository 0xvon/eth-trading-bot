# ETH Trading Bot

This is the repo to trade $ETH automatically.

## Getting Started

0. set env

before run it, edit `src/index.ts` to add your private key.

1. setup

```bash
$ git clone git@github.com:0xvon/eth-trading-bot.git
$ cd eth-trading-bot
$ npm i
```

2. create types from abis

```
$ npm run typechain
```

3. run

```
$ npm start
```
